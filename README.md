# Gremlin APC 710

Hardware design repository for the Gremlin APC 710 project, containing:
- FreeCAD mechanical design files in [CAD](CAD)
- KiCAD electronics design files in [EDA](EDA)

## Repository Structure

- [CAD](CAD)
  - FreeCAD source models (`.FCStd`)
  - Mechanical export files (for example `.step`)
- [EDA](EDA)
  - KiCAD schematic (`.kicad_sch`)
  - KiCAD PCB layout (`.kicad_pcb`)
  - KiCAD project files (`.kicad_pro`)

## Tooling

- Mechanical CAD: FreeCAD V1.1.0
- Electronic design: KiCAD V9.0.6

## Version Control Notes

This repository is configured to ignore temporary and machine-local artifacts generated by FreeCAD, KiCAD, and macOS (for example backup files, lock files, cache files, and `.DS_Store`).
